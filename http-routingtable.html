HTTP Routing Table

/bundles | /collections | /doi | /dois | /products
 
/bundles
GET /bundles request all PDS bundles
GET /bundles/{identifier} bundle URN resolver for lidvid, get one bundle
GET /bundles/{identifier}/all Get one bundle by lidvid, or all versions of a bundle by lid
GET /bundles/{identifier}/collections get collections belonging to a given bundle
GET /bundles/{identifier}/collections/all get collections belonging to a given bundle
GET /bundles/{identifier}/collections/latest get collections belonging to a given bundle
GET /bundles/{identifier}/latest bundle URN resolver for lidvid, get one bundle
GET /bundles/{identifier}/products get products belonging to a given bundle
 
/collections
GET /collections request PDS collections
GET /collections/{identifier} collections URN resolver for lidvid
GET /collections/{identifier}/all Get one collection by lidvid, or all versions of a collection by lid
GET /collections/{identifier}/bundles get bundles containing this given collection
GET /collections/{identifier}/latest collections URN resolver for lidvid
GET /collections/{identifier}/products get producs belonging to a given collection
GET /collections/{identifier}/products/all get producs belonging to a given collection
GET /collections/{identifier}/products/latest get producs belonging to a given collection
 
/doi
GET /doi null
POST /doi/submit null
PUT /doi null
 
/dois
GET /dois null
GET /dois/check null
POST /dois null
 
/products
GET /products search across all PDS data products, including bundles, collections, documentation, context and observational products,
GET /products/{identifier} URN resolver for lidvid
GET /products/{identifier}/all Get one product by lidvid, or all versions of a product by lid
GET /products/{identifier}/bundles get bundles containing product
GET /products/{identifier}/collections get collections containing product
GET /products/{identifier}/latest URN resolver for lidvid
